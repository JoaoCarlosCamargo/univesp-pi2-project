Para executar a aplicação:

No CMD:

'virtualenv venv' ou no windows 'python -m virtualenv venv'

venv\scripts\activate

pip install Flask

pip install Flask-Login

pip install django-ckeditor

py app.py

Acessar pelo browser:

http://localhost:5000/




Instruções Git:

. Inicialize um repositório local:
git init
#Isso irá criar um repositório Git local para rastrear as mudanças no seu projeto.

. Adicione os arquivos ao controle de versão:
git add .
Isso adiciona todos os arquivos do seu projeto ao staging area.

. Faça o primeiro commit:
git commit -m "Initial commit"
Substitua "Initial commit" por uma mensagem mais descritiva sobre o seu projeto.

. Conecte o repositório local ao remoto:
git remote add origin https://github.com/JoaoCarlosCamargo/univesp-pi2-project.git


git branch -M main

. Envie o código para o GitHub:
git push -u origin main
Isso enviará (push) o código do seu repositório local para o repositório remoto na branch principal (main).
